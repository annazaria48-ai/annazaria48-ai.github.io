## 1. Главная страница

| Блок | Что размещаем |
|------|--------------|
| Hero-секция | Профессиональное фото (тёплое, с улыбкой, открытая поза) + заголовок-УТП: кто вы, кому помогаете, в чём суть подхода |
| CTA-кнопка | «Записаться на консультацию» — минимум 2 раза: в шапке и в конце страницы |
| Кому я помогаю | Описание типичных ситуаций клиентов (тревога, травмы, отношения, внутренний конфликт, психосоматика, поиск себя) |
| **Три подхода — развёрнутые блоки** | Гипнотерапия / IFS / Арт-терапия — по каждому: что это, как работает, этапы сессии, с чем помогает, научная база. Ссылки «Подробнее» ведут на статьи блога |
| Как методы работают вместе | Описание комбинаций: арт+IFS, арт+гипно, IFS+гипно |
| Как я работаю | 3 шага: знакомство → регулярные сессии → результат |
| Краткое «Обо мне» | 2–3 предложения + ссылка на полную страницу |
| Цифры | Лет практики, клиентов, часов обучения |
| CTA внизу | Запись + контакты |

**Решение:** Отдельных страниц под методы нет. Развёрнутое описание каждого метода — прямо на главной (средней детализации). Глубокий лонгрид — в блоге (для SEO).

## 4. Как проходит терапия

| Этап | Описание |
|------|----------|
| 1. Вы пишете мне | Короткое сообщение — договариваемся о времени |
| 2. Первая консультация | 60 мин: знакомство, исследование запроса, подбор метода, обсуждение формата |
| 3. Регулярные сеансы | 1 раз/неделю, 60 мин. Работа в выбранном методе или комбинации |
| 4. Чек-ап | Каждые 4–6 сеансов — оценка прогресса |
| 5. Завершение | Закрепление результатов, план поддержки |

Плюс: FAQ — ответы на частые вопросы о терапии.

## Общий вайб и рекомендации

- **Тон текста:** тёплый, бережный, от первого лица, без канцелярита
- **Цветовая палитра:** спокойные, природные тона (бежевый, мягкий зелёный, тёплый серый)
- **Типографика:** читаемые шрифты, много воздуха
- **Фото:** профессиональная фотосессия в тёплых тонах
- **Мобильная версия:** обязательная адаптивность (70%+ трафика с мобильных)
- **SEO:** методы раскрыты на главной + глубокие статьи в блоге = двойное покрытие
- **Платформа:** WordPress + Elementor (максимум гибкости и SEO) или Tilda (быстрый запуск)
